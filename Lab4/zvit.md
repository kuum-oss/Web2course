

## Мета роботи

Отримати практичні навички проєктування та створення елементів навігації веб-сторінки за допомогою мови CSS. Навчитися реалізовувати горизонтальні та вертикальні панелі навігації, створювати багаторівневі випадаючі меню без використання JavaScript, а також дослідити сучасні методи стилізації маркованих та нумерованих списків за допомогою псевдоелементів та CSS-лічильників.

---

## Хід роботи

Під час виконання лабораторної роботи макет веб-сторінки «Основи веб-дизайну» було модернізовано: додано горизонтальну верхню панель навігації, вертикальне меню зліва та інформаційні блоки, структуровані за допомогою списків.

Для створення двоколонкового макета застосовано технологію **CSS Flexbox** (`display: flex`), що дозволило розмістити вертикальну панель (`aside`) та основний контент (`main`) у вигляді адаптивної структури.

### Таблиця 1 — CSS властивості для створення навігації та списків

| Властивість                           | Опис застосування у проєкті                                               |
| ------------------------------------- | ------------------------------------------------------------------------- |
| `position: sticky`                    | Використано для фіксації панелей навігації під час прокручування сторінки |
| `display: none / block`               | Керує появою випадаючого меню при наведенні курсора                       |
| `list-style: none`                    | Вимикає стандартні маркери списків браузера                               |
| `counter-reset` / `counter-increment` | Реалізує користувацьку нумерацію списків                                  |

---

### 1. Горизонтальна панель навігації та Dropdown-меню

Горизонтальну панель розміщено у верхній частині сторінки. Для реалізації випадаючого меню використано вкладений блок з абсолютним позиціонуванням, який відображається при наведенні на батьківський пункт.

```css
/* Логіка роботи випадаючого меню */
.dropdown-content {
    display: none; 
    position: absolute;
    top: 70px; 
    background-color: var(--color-white);
    box-shadow: var(--shadow-lg);
}

/* Відображення підменю при наведенні */
.dropdown:hover .dropdown-content {
    display: block;
}
```


---

### 2. Вертикальна панель та стилізація списків

Вертикальну панель навігації розміщено зліва. Вона також використовує `position: sticky`, тому залишається у полі зору користувача під час прокручування сторінки.

Для оформлення контенту створено маркований (`ul`) та нумерований (`ol`) списки. Замість стандартних маркерів використано псевдоелемент `::before`, що дозволило повністю контролювати стиль відображення.

```css
/* Кастомний нумерований список */
.custom-number-list {
    list-style: none; 
    counter-reset: my-awesome-counter;
}

.custom-number-list li {
    counter-increment: my-awesome-counter; 
}

.custom-number-list li::before {
    content: "0" counter(my-awesome-counter);
    position: absolute;
    left: 0;
    color: var(--color-primary);
    font-weight: 700;
}
```

## Висновки

У результаті виконання лабораторної роботи було успішно розширено функціонал веб-сторінки шляхом додавання сучасних навігаційних елементів та стилізованих списків.

1. **Реалізація навігації.** Доведено, що поєднання горизонтального та вертикального меню значно покращує навігацію користувача. Використання `position: sticky` є сучасним рішенням, яке дозволяє зберігати елементи в зоні видимості без порушення потоку документа.
2. **Dropdown без JavaScript.** Практика показала, що інтерактивні меню можуть бути реалізовані лише засобами CSS. Комбінація `:hover` і керування властивістю `display` дозволяє створювати повноцінну взаємодію без скриптів.
3. **Стилізація списків.** Встановлено, що псевдоелементи та CSS-лічильники забезпечують повний контроль над виглядом списків. Це дозволяє створювати унікальні інтерфейси та автоматично підтримувати правильну нумерацію при зміні структури контенту.

Отримані результати відповідають поставленій меті та демонструють практичне застосування сучасних інструментів CSS для створення зручних та логічних інтерфейсів веб-застосунків.
