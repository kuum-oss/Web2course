## Мета роботи

Отримати практичні навички роботи з каскадними таблицями стилів (CSS). Дослідити та застосувати на практиці блокову модель CSS (CSS Box Model), зокрема властивості налаштування фонів (`background`), зовнішніх відступів (`margin`), внутрішніх полів (`padding`) та рамок (`border`). Створити сучасну веб-сторінку за темою «Картинна галерея», розмістивши елементи з урахуванням принципів сучасного веб-дизайну.

## Хід роботи

Під час виконання лабораторної роботи було розроблено веб-сторінку «Віртуальна Картинна Галерея», яка містить шедеври українського та світового мистецтва. Сторінка складається з головного банера (Hero section) та двох логічних секцій з інтерактивними картками картин.

Для стилізації елементів було застосовано зовнішній файл стилів `style.css`. Основний акцент було зроблено на демонстрації можливостей CSS щодо керування відступами та фонами, згідно з правилами Box Model. Короткий опис застосованих властивостей наведено в таблиці нижче.

### Таблиця 1 — Основні використані CSS властивості для блокової моделі

| Властивість    | Опис застосування у проєкті                                                                                              | Приклад елемента                                      |
| -------------- | ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------- |
| **Background** | Використано суцільні кольори, градієнти (`linear-gradient`), фонові зображення (`url`) та напівпрозорі заливки (`rgba`). | Головний банер (`.gallery-hero`), підкладка карток    |
| **Border**     | Створено суцільні рамки (`solid`), налаштовано товщину та колір. Застосовано заокруглення кутів (`border-radius`).       | Рами картин (`.artwork`), скляний блок (`.hero-box`)  |
| **Margin**     | Застосовано для центрування блоків (`margin: 0 auto`) та створення простору між незалежними елементами.                  | Відступи між секціями та заголовками                  |
| **Padding**    | Використано для формування внутрішнього простору елементів, створення ефекту паспарту для картин.                        | Внутрішні поля карток (`.art-info`) та рам (`.frame`) |

Для створення візуального ефекту музейного оформлення картин був використаний комплексний підхід до налаштування відступів та рамок. Нижче наведено фрагмент програмного коду, який демонструє стилізацію зображення та його обгортки.

```css
/* Імітація паспарту (рами картини) за допомогою padding та background */
.frame {
    padding: 30px;
    background-color: #faf9f6;
    border-bottom: 2px solid #eaeaea;
}

/* Налаштування самого зображення за допомогою border */
.artwork {
    width: 100%;
    height: 300px;
    object-fit: cover;
    border: 8px solid #222;
    border-radius: 4px;
    box-shadow: inset 0 0 20px rgba(0,0,0,0.5), 0 5px 15px rgba(0,0,0,0.3);
}
```

Зовнішній вигляд створеної віртуальної галереї представлено нижче.



Стилізація секції карток була виконана з використанням технології CSS Grid Layout, що дозволило адаптивно розмістити елементи. Візуалізація відступів (`margin`) між картками та внутрішніх полів (`padding`) наведена нижче.

## Висновки

У результаті виконання лабораторної роботи було успішно спроєктовано та розроблено веб-сторінку «Віртуальна Картинна Галерея». Було виконано детальний аналіз та практичне застосування блокової моделі CSS (CSS Box Model).

Оцінюючи виконану роботу відповідно до поставленої мети, можна констатувати наступне:

1. **Теоретичні положення підтверджено практикою.** Робота підтвердила, що будь-який HTML-елемент є прямокутним блоком. Комбінування властивостей `padding` та `border` дозволило створити ефект «картини в рамі з паспарту» без використання додаткових графічних редакторів.
2. **Керування фонами.** Використання множинних фонів (поєднання `linear-gradient` із `url()`), а також властивості `background-attachment: fixed` дозволило досягти сучасного ефекту паралаксу та забезпечити високу контрастність тексту на головному екрані.
3. **Сучасний дизайн.** Завдяки використанню напівпрозорих фонів (`rgba`), тіней (`box-shadow`) та анімацій наведення (`hover`, `transform: translateY`) отриманий інтерфейс відповідає сучасним тенденціям UI/UX дизайну.

Завдання лабораторної роботи виконані в повному обсязі. Набуті навички дозволяють ефективно керувати позиціонуванням елементів та їх візуальним оформленням за допомогою каскадних таблиць стилів.
