## МЕТА РОБОТИ

Ознайомитися з можливостями мови CSS3 у проєктуванні динамічних елементів вебсайтів. Вивчити та застосувати на практиці правила створення анімацій (`@keyframes`), плавних переходів (`transition`) і стилізації різних типів кнопок керування. Адаптувати створену раніше вебсторінку відповідно до сучасних принципів інтерактивного дизайну.

## ХІД РОБОТИ

У ході виконання лабораторної роботи до раніше створеного макета **Premium Art Gallery** було впроваджено CSS-анімації та сучасні інтерактивні кнопки. Це дозволило зробити інтерфейс більш динамічним і покращити користувацький досвід (UX).

Для реалізації поставлених завдань застосовувалися властивості CSS3, наведені у таблиці.

### Таблиця 1 — Основні CSS-властивості для створення анімацій і кнопок

| Властивість       | Опис застосування                                                        |
| ----------------- | ------------------------------------------------------------------------ |
| `@keyframes`      | Створення користувацьких сценаріїв анімації (поява елементів, пульсація) |
| `animation`       | Застосування ключових кадрів до HTML-елементів із зазначенням тривалості |
| `animation-delay` | Затримка перед початком анімації, використана для каскадної появи карток |
| `transition`      | Плавна зміна станів кнопок під час наведення курсора                     |
| `transform`       | Масштабування та зміщення елементів інтерфейсу                           |

## 1. Створення CSS-анімацій

Для привернення уваги користувача під час завантаження сторінки було реалізовано анімацію плавної появи елементів знизу вгору. Використання `animation-delay` з різними значеннями дозволило досягти ефекту каскадної появи карток.

```css
@keyframes fadeInUp {
    from { 
        opacity: 0; 
        transform: translateY(40px); 
    }
    to { 
        opacity: 1; 
        transform: translateY(0); 
    }
}

.animate-up {
    animation: fadeInUp 0.8s cubic-bezier(0.2, 0.8, 0.2, 1) forwards;
    opacity: 0;
}

.delay-1 { animation-delay: 0.2s; }
.delay-2 { animation-delay: 0.4s; }
```

## 2. Стилізація кнопок

Було розроблено три типи кнопок:

1. Залита кнопка (Solid)
2. Контурна кнопка (Outline)
3. Пульсуюча кнопка (Call to Action)

Для пульсуючої кнопки застосовано безкінечну анімацію світіння з використанням `box-shadow`.

```css
@keyframes glowPulse {
    0% { box-shadow: 0 0 0 0 rgba(212, 175, 55, 0.6); transform: scale(1); }
    50% { box-shadow: 0 0 0 20px rgba(212, 175, 55, 0); transform: scale(1.05); }
    100% { box-shadow: 0 0 0 0 rgba(212, 175, 55, 0); transform: scale(1); }
}

.btn-pulse {
    background-color: var(--gold-accent);
    color: var(--bg-main);
    animation: glowPulse 2s infinite; 
}
```

## ВИСНОВКИ

У результаті виконання лабораторної роботи було модернізовано вебсторінку за рахунок інтеграції CSS3-анімацій та інтерактивних кнопок.

1. **Продуктивність:** складні візуальні ефекти можуть бути реалізовані засобами CSS без використання JavaScript, що зменшує навантаження на браузер.
2. **UX:** застосування `transition`, `transform` і тіней забезпечує плавну реакцію інтерфейсу та покращує його сприйняття користувачем.
3. **Сучасні практики:** використання анімації світіння для CTA-кнопок відповідає актуальним тенденціям вебдизайну та спрямовує увагу користувача на ключові дії.

Поставлену мету лабораторної роботи досягнуто повністю. Отримані навички дозволяють створювати сучасні інтерактивні інтерфейси з високим рівнем залучення користувача.
