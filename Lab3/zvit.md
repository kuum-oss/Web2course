## Мета роботи

Дослідити та засвоїти на практиці можливості мови CSS щодо роботи з типографікою (підключення та налаштування шрифтів), застосування блочної моделі (Box Model) та контурів (Outline). Створити веб-сторінку за темою «Веб-дизайн для початківців» з використанням сучасних принципів та тенденцій UI/UX дизайну.

---

## Хід роботи

Під час виконання лабораторної роботи було розроблено сучасну веб-сторінку, яка демонструє ключові принципи CSS. Інтерфейс побудовано з використанням глибоких тіней (`box-shadow`), градієнтів та заокруглень (`border-radius`), що відповідає актуальним тенденціям веб-дизайну.

Особливу увагу було приділено типографіці. Згідно із завданням, для забезпечення візуальної ієрархії було підключено 4 різні гарнітури з бібліотеки Google Fonts. Розподіл шрифтів наведено у таблиці нижче.

### Таблиця 1 — Використання шрифтів на веб-сторінці

| Назва шрифту (Гарнітура) | Тип шрифту               | Призначення у проєкті                                             |
| ------------------------ | ------------------------ | ----------------------------------------------------------------- |
| **Oswald**               | Без засічок (Sans-serif) | Головні заголовки (h1, h2, h3), привернення уваги                 |
| **Merriweather**         | Із засічками (Serif)     | Акценти, підзаголовки та блоки з цитатами                         |
| **Roboto**               | Без засічок (Sans-serif) | Основний масив тексту для забезпечення максимальної читабельності |
| **Fira Code**            | Моноширинний (Monospace) | Виділення фрагментів програмного коду та технічних термінів       |

Керування шрифтами та кольоровою палітрою реалізовано через CSS-змінні (Custom Properties) у кореневому селекторі `:root`, що дозволяє легко масштабувати проєкт.

```css
/* Налаштування шрифтів через CSS-змінні */
:root {
    --font-heading: 'Oswald', sans-serif;
    --font-serif: 'Merriweather', serif;
    --font-body: 'Roboto', sans-serif;
    --font-code: 'Fira Code', monospace;
    --color-primary: #4f46e5;
}

body {
    font-family: var(--font-body);
    line-height: 1.75; 
    font-size: 17px;   
}
```

Зовнішній вигляд секції, що демонструє роботу зі шрифтами, наведено нижче.


Друга частина роботи була присвячена дослідженню блочної моделі (Box Model) та властивості `outline`. Для наочності було створено інтерактивні картки. Блок демонстрації контуру використовує псевдокласи `:hover` та `:focus-within` для створення ефекту світіння без зміни фізичних розмірів елемента.

```css
/* Інтерактивний контур (Outline) при фокусі або наведенні */
.outline-demo:hover, .outline-demo:focus-within {
    outline: 3px solid var(--color-primary);
    outline-offset: 6px; /* Відступ контуру від межі елемента */
    box-shadow: 0 0 0 10px rgba(79, 70, 229, 0.1);
}
```

Результат роботи блочної моделі та контурів представлено нижче.

## Висновки

У ході виконання лабораторної роботи було успішно спроєктовано та розроблено веб-сторінку «Основи сучасного веб-дизайну». Порівнюючи отримані результати з теоретичними положеннями, можна зробити наступні висновки:

1. **Ефективність типографіки.** Практично доведено, що використання різних типів шрифтів (засічкові, без засічок, моноширинні) суттєво покращує сприйняття інформації. Використання змінних для підключення кількох шрифтів дозволило створити чітку ієрархію без перевантаження коду. Налаштування `line-height` та `letter-spacing` забезпечило відповідність макета сучасним стандартам читабельності.
2. **Особливості Box Model.** Робота підтвердила теоретичні положення про те, що загальний розмір елемента складається з його контенту, внутрішніх полів (`padding`) та рамки (`border`). Використання глобального скидання `box-sizing: border-box;` дозволило уникнути проблем із неочікуваним збільшенням ширини блоків при додаванні внутрішніх відступів.
3. **Відмінність Outline від Border.** Практичне тестування довело, що властивість `outline` не впливає на фізичні розміри блоку та не зміщує сусідні елементи у потоці документа. Використання `outline-offset` у поєднанні з м'якими тінями дозволило створити сучасний ефект фокусування на елементі, що покращує доступність сторінки при навігації з клавіатури.

Отримані результати повністю відповідають поставленій меті. Набуті навички є базовими для подальшої професійної верстки сучасних веб-інтерфейсів.
