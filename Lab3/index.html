1. Магія Типографіки

Типографіка — це 95% веб-дизайну. Вона створює настрій та ієрархію.

У сучасному дизайні шрифти — це головний інструмент. На цій сторінці ми комбінуємо чотири різні гарнітури для створення візуального ритму:

  • Заголовки Oswald — Масивний, привертає увагу.
  • Акценти Merriweather — Елегантний, для цитат.
  • Основний текст Roboto — Чистий, легко читається.
  • Код Fira Code — Для технічних термінів, наприклад <div> або color: #ff6b6b;.

"Хороший дизайн — це якнайменше дизайну. Це не про те, як воно виглядає, а про те, як воно працює."

— Стів Джобс (адаптовано для вебу)

Box Model у дії

Цей блок демонструє класичну модель:
Margin (прозора зона зовні),
Border (кольорова рамка),

Інтерактивний Контур (Outline)

Наведіть мишку або натисніть Tab.
На відміну від рамки (Border), outline не займає місця в потоці документу і часто використовується для фокусу.
