## ХІД РОБОТИ

Під час виконання лабораторної роботи було розроблено дизайн-макет багатосторінкового інформаційного веб-сайту **«Природні скарби планети»**. Проєкт складається з трьох пов’язаних HTML-сторінок (`index.html`, `parks.html`, `wonders.html`), які використовують єдиний файл стилів `style.css`.

Відповідно до теоретичних положень, макет кожної сторінки було розділено на логічні блоки:

1. **Header (Заголовок)** — містить назву проєкту та фонове зображення.
2. **Navigation Menu (Меню навігації)** — забезпечує перехід між сторінками.
3. **Main Content (Основний контент)** — поділений на дві колонки: основні матеріали зліва та додаткова інформація справа.
4. **Footer (Підвал)** — містить копірайт і службову інформацію.

Для створення двоколонкового макета замість застарілої властивості `float` було використано сучасну технологію **CSS Flexbox**. Основні властивості CSS, застосовані у проєкті, наведені в таблиці.

### Таблиця 1 — CSS-властивості для побудови структури макета

| Властивість              | Опис застосування у проєкті                                                                  |
| ------------------------ | -------------------------------------------------------------------------------------------- |
| `box-sizing: border-box` | Дозволяє включати padding і рамки в загальну ширину елемента, що запобігає руйнуванню макета |
| `display: flex`          | Застосовано до контейнера (`.row`) для розміщення колонок в один ряд                         |
| `flex: 3` / `flex: 1`    | Пропорційний розподіл ширини між лівою (75%) і правою (25%) колонками                        |
| `@media screen`          | Забезпечує адаптивність макета на мобільних пристроях                                        |

Нижче наведено фрагмент коду, що відповідає за структуру колонок і адаптивність макета:

```css
/* Побудова колонок за допомогою Flexbox */
.row {
    display: flex;
    flex-wrap: wrap;
    padding: 20px;
    gap: 20px;
}

.leftcolumn {
    flex: 3;
    min-width: 300px;
}

.rightcolumn {
    flex: 1;
    min-width: 250px;
}

/* Адаптивність для мобільних екранів */
@media screen and (max-width: 800px) {
    .row {
        flex-direction: column;
    }
}
```

Для меню навігації було використано властивість `position: sticky; top: 0;`, що дозволяє меню залишатися у верхній частині екрана під час прокручування сторінки. Перехід між сторінками реалізовано через зміну класу `.active` для відповідного посилання на кожній сторінці.

## ВИСНОВКИ

У результаті виконання лабораторної роботи було спроєктовано та реалізовано макет багатосторінкового веб-сайту.

1. **Єдиний стиль сайту:** використання одного файлу стилів `style.css` для трьох HTML-сторінок підтвердило ефективність каскадних таблиць стилів і дозволило уникнути дублювання оформлення.
2. **Переваги сучасної верстки:** застосування `CSS Flexbox` дозволило швидко створити двоколонкову структуру без використання застарілих підходів (`float`, clearfix).
3. **Адаптивність:** використання медіа-запитів забезпечило коректне відображення сайту на різних пристроях. Перехід контейнера в режим `flex-direction: column` дозволяє сторінці зручно відображатися на мобільних екранах.

Отже, поставлену мету лабораторної роботи досягнуто повністю. Розроблений макет є структурно правильним, адаптивним, зручним у навігації та відповідає сучасним стандартам веб-верстки.
