## МЕТА РОБОТИ

Ознайомитися з можливостями мови CSS3 у проєктуванні динамічних елементів веб-сайтів. Вивчити та застосувати на практиці правила створення анімацій (`@keyframes`), плавних переходів (`transition`) та стилізації різних типів кнопок управління. ## ХІД РОБОТИ

Під час виконання лабораторної роботи у раніше створений макет **«Premium Art Gallery»** було впроваджено CSS-анімації та сучасні інтерактивні кнопки. Це дозволило зробити інтерфейс більш динамічним та покращити користувацький досвід (UX).

Для реалізації поставлених завдань застосовувалися властивості CSS3, опис яких наведено у таблиці.

### Таблиця 1 — Основні CSS-властивості для створення анімацій та кнопок

| Властивість       | Опис застосування у проєкті                                              |
| ----------------- | ------------------------------------------------------------------------ |
| `@keyframes`      | Створення користувацьких сценаріїв анімації (поява елементів, пульсація) |

### 1. Створення CSS-анімацій

Для привернення уваги користувача під час завантаження сторінки реалізовано анімацію плавної появи елементів знизу вгору. Використання властивості `animation-delay` з різними значеннями дозволило створити ефект каскадної появи тексту головного екрана та карток сервісів.

```css
/* Сценарій анімації появи (Fade In Up) */
@keyframes fadeInUp {
    from { opacity: 0; transform: translateY(40px); }
    to { opacity: 1; transform: translateY(0); }
}

/* Застосування анімації до елементів */
.animate-up {
    animation: fadeInUp 0.8s cubic-bezier(0.2, 0.8, 0.2, 1) forwards;
    opacity: 0;
}
.delay-1 { animation-delay: 0.2s; }
.delay-2 { animation-delay: 0.4s; }
```

Такий підхід дозволяє зробити сторінку більш «живою» без використання JavaScript.

### 2. Стилізація кнопок

У межах лабораторної роботи було реалізовано три типи кнопок:

1. **Залита кнопка (`.btn-gold`)** — використовується для основних дій.
2. **Контурна кнопка (`.btn-outline`)** — застосовується для другорядних дій у картках.
3. **Пульсуюча кнопка (`.btn-pulse`)** — використовується як акцентний елемент Call to Action.

Для пульсуючої кнопки застосовано нескінченну анімацію світіння з використанням властивості `box-shadow`.

```css
/* Анімація пульсації для кнопки */
@keyframes glowPulse {
    0% { box-shadow: 0 0 0 0 rgba(212, 175, 55, 0.6); transform: scale(1); }
    50% { box-shadow: 0 0 0 20px rgba(212, 175, 55, 0); transform: scale(1.05); }
    100% { box-shadow: 0 0 0 0 rgba(212, 175, 55, 0); transform: scale(1); }
}

.btn-pulse {
    background-color: var(--gold-accent);
    color: var(--bg-main);
    animation: glowPulse 2s infinite;
}
```

Застосування такої кнопки дозволяє інтуїтивно спрямувати увагу користувача на ключову дію.

У результаті виконання лабораторної роботи було модернізовано веб-сторінку шляхом інтеграції CSS3-анімацій та стилізованих кнопок.

3. **Відповідність сучасним тенденціям:** використання акцентної пульсуючої кнопки відповідає практикам сучасного веб-дизайну та допомагає виділити ключову дію.

Отже, поставлену мету лабораторної роботи досягнуто. Отримані навички дозволяють створювати сучасні інтерактивні веб-інтерфейси з високим рівнем залучення користувача.
